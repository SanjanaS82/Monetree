# Monetree 
A personal finance management web application built using **React + TypeScript + Vite + TailwindCSS**.

## Features

*  Deposit tracking
*  Savings progress with visual progress bar
*  Lock/Unlock feature for funds
*  Tab-based UI for better navigation

## Tech Stack

* React (with TypeScript)
* Vite
* Tailwind CSS
* shadcn/ui components

## Project Structure

```
Monetree/
├── public/            # Static assets
├── src/               # Main application code
│   ├── components/    # Reusable UI components
│   ├── pages/         # Tab pages (Deposit, Savings, Lock, etc.)
│   ├── App.tsx        # Root component
│   └── main.tsx       # Entry point
├── index.html
├── package.json
├── tailwind.config.ts
├── vite.config.ts
└── tsconfig.json
```

## Run in local

### 1. Clone the repository

```bash
git clone https://github.com/<your-username>/Monetree.git
cd Monetree
```

### 2. Install dependencies (using npm)

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
